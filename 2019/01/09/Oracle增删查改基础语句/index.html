Oracle增删查改基础语句

发表于 2019-01-09 | | 阅读次数:

Oracle增加数据

Ps. 下面语句中用   [   ]   括起来的即可以省略的部分

1.insert语句

用于插入单行数据

语法

insert   [into]   <表名>   [列名]   values   <列值>;
eg.往表格a里插入一行姓名为齐天大圣，性别男，出生日期为1999-10-12的数据

insert a (姓名,性别,出生日期) values (‘齐天大圣’,’男’,’1999/10/12’);

Ps. 如果表a只有这三个列名，也可以省略列名，后面的values就会依次排序自动插入

2.insert select语句

把旧表格数据添加到新表格

语法

insert   into   <新表名>   <列名>   select   <已有表格列名>   from   <已有表名>;
eg.把旧表格的姓名，性别，地址数据复制到新表格

insert into table_new (‘姓名’,’性别’,’地址’) select name,sex,address from table_old;

Oracle删除数据

delete语句

删除表格里的某些数据

语法

delete   from   <表名[where<删除条件>];
eg.删除表格a中名为天天的行

delete from a where name = ‘天天’;

Ps.该方法为直接删除名为天天的整行数据

truncate table语句

语法

truncate table   <表名>;
eg.删除表a

truncate table a;

Ps.表格a所有行的数据将被删除，但是保留表格a的结构、列名等

Oracle修改数据

update语句

语法

update   <表名>   set   <列名=更新值>   [where<更新条件>];
eg.修改表a里天天的年龄为18

update a set age = 18 where name = ‘天天’;

Ps.set后可跟多个数据列进行更新数据

Oracle查询数据

查询数据为Oracle增删查改最重要的一点

普通查询

语法

select * from 表名;        查询表格所有列的数据
select column from 表名;       查询指定列的数据
eg.查询表a全体学生的学号和姓名

select sid,name from a;

若查询结果有重复行，可使用distinct
eg.查询表a全体学生的学号和姓名，排除重复行

select distinct sid,name from a;

查询日期

日期列：是指数据类型为日期类型的列，如date类型
日期列的默认显示格式为DD-MON-RR，RR代表年份，而且只取年份的后两位数字
如果以简体中文显示日期结果，那么可以将会话的nls_language参数设置为simplified chinese
eg.设置简体中文日期格式

alter session set nls_language = ‘SIMPLIFIED CHINESE’;

eg.设置美国日期格式

alter session set nls_language = ‘AMERICAN’;

eg.设置特定日期格式

alter session set nls_date_format = ‘YYYY/MM/DD’;

算术运算符

主要应用于数字和日期的列上，优先级为乘除，再到加减，有括号先算括号里面的
eg.查询表格a的年工资（原表格a只有月工资sal的基础上）

select sal*12 from a;

判断空值   is null   /   is not null

用于检测列或表达式的结果是否为null，如果为null，则返回true，否则返回false
eg.判断表a姓名列是否为null

select id,name,sex from a where name is null;
select id,name,sex from a where name is not null;

order by语句排序   asc（默认）  /   desc

order by语句和其他语句一起使用时，必须位于最后，分组时不能使用where语句，应使用having语句
eg.从表格a选择姓名和电话，按照工资升序排列

select name,cal from a order by sal asc;
select name,cal from a order bt sal;     默认升序

eg.从表格a选择姓名和电话，按照工资降序排列

select name,cal from a order by sal desc;

eg.使用别名排序，从表格a选择姓名和电话，按照年收入升序降序排列

select name,cal,sal*12 年收入 from a order by 年收入 desc;

多列排序，首先按照第一列进行排序，相同的以第二列数据进行排序，以此类推；
eg.从表格a选择姓名和电话，按照工资升序、年龄降序排列

select name,cal from a order by sal asc,age desc;

模糊查询

like语句

当使用like语句进行模糊查询时，需要使用通配符%和_
通配符%：用于表示0个或多个字符
通配符_:用于表示单个字符
eg.通配符%，模糊查询表格a名字第一个字符是H后面任意

select from a where name like ‘H%’;
eg.通配符_，模糊查询表格a名字第一个字符任意，中间是RT，后面任意
select from a where name like ‘_RT%’;

between语句

between即在某个范围内进行查询
eg.查找表格a里年龄介于18到22之间的人

select * from a where age between 18 to 22;

in语句

满足in()里其中一个
eg.查询表a仲address值为广东或者上海或者香港得记录，显示name字段

select name from a where address in (‘广东’,’上海’,’香港’);

having语句

由于where只能在没有分组时使用，分组后只能使用having来限制条件
eg.查询表a里count(score)>1的行，输出学号和平均成绩，按照学号进行升序排列

select id as 学号, AVG(score) as 平均成绩 from a group by id having count(score)>1;

多表连接查询

left join：左连接，返回左表中所有的记录以及右表中连接字段相等的记录
right join：右连接，返回右表中所有的记录以及左表中连接字段相等的记录
inner join：内连接，又叫等值连接，只返回两个表中连接字段相等的行
full join：外连接，返回两个表中所有的行：left join + right join
cross join：结果是笛卡尔积，即第一个表的行数乘以第二个表的行数

内连接

在where子句中指定连接条件
eg.查询表a和表b中name字段相等的记录，并显示表a中的name字段和表b中的score字段

select a.name,b.score from a,b where a.name = b.name;

eg.采用内连接查询表a和表b中name字段相等的记录，并显示表a中的name字段和表b中的score字段

select a.name,b.score from a inner join b on a.name=b.name;

外连接

left/right/full outer join ， 通常省略outer关键字，写成：left/right/full join
在左连接和右连接时都会以一张表a为基础表，该表的内容会全部显示，再加上表a和表b匹配的内容，如果表a的数据在表b中没有记录。那么在相关联的结果集行中列显示为空值null

左外连接查询

eg.采用左外连接查询表a中code和表b中id相等的记录，并显示表a中的name字段和表b中的id字段、score字段

select a.name,b.id,b.score from a left join b on a.code = b.id;

右外连接查询

eg.采用右外连接查询表a中code和表b中id相等的记录，并显示表a中的name字段和表b中的id字段、score字段

select a.name,b.id,b.score from a right join b on a.code = b.id;

复杂查询

数据分组

max,min,avg,sum,count
eg.显示表a所有员工中最高工资和最低工资

select max(sal),min(sal) from a;

eg.显示表a所有员工中最高工资和最高工资的人是谁

select name,sal from a where sal = (select max(sal) from a);

eg.显示表a工资高于平均工资的员工信息

select * from a where sal > (select avg(sal) from a);

group by 和 having 子句

eg.显示表a每个部门的平均工资和最高工资

select avg(sal),max(sal),deptno from a gruop by deptno;

eg.显示表a每个部门的每种岗位的平均工资和最高工资

select avg(sal),max(sal),deptno,job from a gruop by deptno,job;

eg.显示表a平均工资低于2000的部门号和它的平均工资

select deptno,avg(sal) from a group by deptno having avg(sal) < 2000;

数据分组总结

分组函数只能出现在选择列表、having、order by子句中
如果在select语句中同时包含有group by、having、order by那么他们的优先级是group by,having,order by
在选择列中如果有列、表达式和分组函数，那这些列和表达式必须有一个出现在group by子句中

SQL函数的使用

字符函数

lower(char)：将字符串转化为小写的格式
upper(char)：将字符串转化为大写的格式
length(char)：返回字符串的长度
substr(char,m,n)：取字符串的子串， m 表示从第几个开始， n 表示取几个字符
eg.以首字母大写的方式显示表a所有员工的姓名(name);组合使用上面 4 个函数

select upper(substr(name,1,1)) || lower(substr(name,2,length(name)-1)) from a;

replace(char1,s_str,r_str)：替换字符串
eg.使用 “天天” 替换表a员工姓名（name）里的 “A”

select replace(name,’A’,’天天’) from a;

数学函数（财务或银行系统常用）

round(n,[m])：四舍五入，省略 m 则四舍五入到整数；m 是正数则四舍五入到小数点的m位后；m 是负数则四舍五入到小数点的 m 位前
trunc(n,[m])： 截取数字，省略 m 即截去小数部分；m 是正数截取到小数点的 m 位后；m 是负数截取到小数点的前 m 位
mod(m,n)：取余数
floor(n)：返回小于或是等于 n 的最大整数
ceil(n)：返回大于或是等于 n 的最小整数
eg.显示表 a 在一个月为 30 天的情况所有员工的日薪金（sal），忽略余数

select trunc(sal/30),name from a;
select floor(sal/30),name from a;

abs(n)：返回数字 n 的绝对值
acos(n)：返回数字的反余弦值
asin(n)：返回数字的反正弦值
atan(n)：返回数字的反正切
cos(n)：返回数字的余弦值
exp(n)：返回 e 的 n 次幂
log(m,n)：返回对数值
power(m,n)：返回 m 的 n 次幂

日期函数

Oracle 默认日期格式是 ‘dd-mon-yy’
sysdate：返回系统时间
add_months(d,n)：在表格原始数据上增加月份
last_day(d)：返回指定日期所在月份的最后一天
eg.查找表a已经入职（入职时间 hiredate） 8 个月以上的员工

select * from a where sysdate > add_months(hiredate,8);

eg.显示表a满10年服务年限的员工的姓名（name）合受雇日期（hiredate）

select name,hiredate from a where sysdate >= add_months(hiredate,12*10);

eg.对于表a每个员工，显示其加入公司的天数

select trunc(sysdate-hiredate) AS 入职天数,name from a;

eg.找出表a各月份倒数第3天受雇（受雇日期 hiredate）的所有员工

select hiredate,name fromm a where last_day(hiredate) - 2 = hiredate;

转换函数

to_char：to_char(hiredate,’YYYY-mm-dd hh24:mi:ss’); 即把原来的日期格式转换为’YYYY-mm-dd hh24:mi:ss’格式
