Oracle增删查改基础语句

发表于 2019-01-09 | | 阅读次数:

Oracle 增加数据

Ps. 下面语句中用   [   ]   括起来的即可以省略的部分

insert 语句

用于插入单行数据

Ps. 如果表a只有这三个列名，也可以省略列名，后面的 values 就会依次排序自动插入

insert select 语句

把旧表格数据添加到新表格

Oracle 删除数据

delete 语句

删除表格里的某些数据

Ps.该方法为直接删除名为天天的整行数据

truncate table 语句

Ps.表格 a 所有行的数据将被删除，但是保留表格 a 的结构、列名等;

truncate 与 delete 区别

  • truncate 是 DDL 语句， delete 是 DML 语句
  • truncate 比 delete 删除速度快，效率高，但是 truncate 语句不能带 where 条件，只能删除全部数据，而 delete 可删除部分数据
  • truncate 是永久删除不能回滚，而 delete 可以使用 rollback 命令进行数据回滚
  • 如果由 foreign key 约束引用的表，不能使用 truncate table , 需要使用不带 where 子句的 delete 语句

drop table 语句

purge

可选，不带 purge 选项，表示删除的表放进回收站，空间不回收，使用 flashback 可以闪回该表
带 purge 选项，则表示将表删除，释放空间

cascade constraints

可选，如果有其他表关联到要删除的表的主键列或唯一键列，直接用 drop table 会报错，若强制删除该表，可加上 cascade constraints

Oracle 修改数据

update 语句

Ps. set 后可跟多个数据列进行更新数据

Oracle查询数据

查询数据为 Oracle 增删查改最重要的一点

普通查询

若查询结果有重复行，可使用distinct

查询日期

日期列：是指数据类型为日期类型的列，如 date 类型
日期列的默认显示格式为 DD-MON-RR , RR代表年份，而且只取年份的后两位数字
如果以简体中文显示日期结果，那么可以将会话的 nls_language 参数设置为 simplified chinese

算术运算符

主要应用于数字和日期的列上，优先级为乘除，再到加减，有括号先算括号里面的

判断空值   is null   /   is not null

用于检测列或表达式的结果是否为 null, 如果为 null, 则返回 true, 否则返回 false

order by 语句排序   asc（默认）  /   desc

order by 语句和其他语句一起使用时，必须位于最后，分组时不能使用 where 语句，应使用 having 语句

多列排序，首先按照第一列进行排序，相同的以第二列数据进行排序，以此类推

模糊查询

like 语句

当使用 like 语句进行模糊查询时，需要使用通配符 % 和 _
通配符%：用于表示 0 个或多个字符
通配符_:用于表示单个字符

若要查询的字符过长，可使用 substr()

Ps. dual 是 Oracle 中的一个实际存在的表，任何用户均可读取，常用在没有目标表的 select 语句块中

between 语句

between 即在某个范围内进行查询

in 语句

满足 in() 里其中一个

having 语句

由于 where 只能在没有分组时使用，分组后只能使用 having 来限制条件

where 语句与 having 语句的区别

  • where 不能放在 group by 后面
  • having 跟在 group by 后面，此时作用相当于 where
  • where 后面的条件中不能有聚集函数，比如 sum() , avg() 等， having 则可以
  • 聚合语句（sum , avg）比 having 子句优先执行，where 子句优先于聚合语句

多表连接查询

left join：左连接，返回左表中所有的记录以及右表中连接字段相等的记录
right join：右连接，返回右表中所有的记录以及左表中连接字段相等的记录
inner join：内连接，又叫等值连接，只返回两个表中连接字段相等的行
full join：外连接，返回两个表中所有的行：left join + right join
cross join：结果是笛卡尔积，即第一个表的行数乘以第二个表的行数

内连接

在 where 子句中指定连接条件

外连接

left / right / full outer join ， 通常省略 outer 关键字，写成：left / right / full join
在左连接和右连接时都会以一张表 a 为基础表，该表的内容会全部显示，再加上表 a 和表 b 匹配的内容，如果表 a 的数据在表 b 中没有记录，那么在相关联的结果集行中列显示为空值 null

数据分组

max,min,avg,sum,count

group by 和 having 子句

数据分组总结

分组函数只能出现在选择列表、having、order by 子句中
如果在select语句中同时包含有 group by、having、order by 那么他们的优先级是 group by,having,order by
在选择列中如果有列、表达式和分组函数，那这些列和表达式必须有一个出现在 group by 子句中

SQL函数的使用

字符函数

lower(char)：将字符串转化为小写的格式
upper(char)：将字符串转化为大写的格式
length(char)：返回字符串的长度
substr(char,m,n)：取字符串的子串， m 表示从第几个开始， n 表示取几个字符

replace(char1,s_str,r_str)：替换字符串

数学函数（财务或银行系统常用）

round(n,[m])：四舍五入，省略 m 则四舍五入到整数；m 是正数则四舍五入到小数点的 m 位后；m 是负数则四舍五入到小数点的 m 位前
trunc(n,[m])： 截取数字，省略 m 即截去小数部分；m 是正数截取到小数点的 m 位后；m 是负数截取到小数点的前 m 位
mod(m,n)：取余数
floor(n)：返回小于或是等于 n 的最大整数
ceil(n)：返回大于或是等于 n 的最小整数

abs(n)：返回数字 n 的绝对值
acos(n)：返回数字的反余弦值
asin(n)：返回数字的反正弦值
atan(n)：返回数字的反正切
cos(n)：返回数字的余弦值
exp(n)：返回 e 的 n 次幂
log(m,n)：返回对数值
power(m,n)：返回 m 的 n 次幂

日期函数

Oracle 默认日期格式是 ‘dd-mon-yy’
sysdate：返回系统时间
add_months(d,n)：n 为正数时，在表格原始数据上增加月份；若 n 为负数时，则在表格原始数据上减少月份
last_day(d)：返回指定日期所在月份的最后一天

转换函数

to_char 格式化函数提供一套有效的工具用于把各种数据类型（日期/时间，int,float,numeric）转换成格式化的字符串以及反过来从格式化的字符串转换成原始的数据类型
to_char(hiredate,’YYYY-mm-dd hh24:mi:ss’); 即把原来的日期格式转换为’YYYY-mm-dd hh24:mi:ss’格式
to_char(int,text);即把 int4/int8 转换成 string

to_date：用于将字符串转换成 date 类型的数据
